## 🚀 現在開始部署（只需 3 個步驟）

### 步驟 1：建立 GitHub 儲存庫（5 分鐘）

1. **前往 GitHub**
   - 開啟瀏覽器訪問：https://github.com
   - 如果沒有帳號，請先註冊（免費）

2. **建立新儲存庫**
   - 登入後，點擊右上角的 `+` 號
   - 選擇 `New repository`
   - 填寫資訊：
     * Repository name: `toeic-vocab-test`
     * Description: `TOEIC 單字測驗網頁應用`
     * 選擇 `Public`（公開）
     * **不要**勾選 "Add a README file"
   - 點擊 `Create repository`

3. **複製儲存庫網址**
   - 在新建立的儲存庫頁面，找到網址（類似這樣）：
     ```
     https://github.com/YOUR_USERNAME/toeic-vocab-test.git
     ```
   - 複製這個網址

4. **上傳程式碼**
   - 在 PowerShell 中執行以下命令（替換網址為你的）：
   
   ```powershell
   # 連接到你的 GitHub 儲存庫
   git remote add origin https://github.com/YOUR_USERNAME/toeic-vocab-test.git
   
   # 推送程式碼
   git branch -M main
   git push -u origin main
   ```

   - 如果要求輸入帳號密碼，請輸入你的 GitHub 帳號和密碼
   - （現在可能需要使用 Personal Access Token，而不是密碼）

---

### 步驟 2：部署到 Render.com（3 分鐘）

1. **註冊 Render**
   - 前往：https://render.com
   - 點擊 `Get Started for Free`
   - **建議使用 GitHub 帳號登入**（這樣更方便）

2. **建立新服務**
   - 登入後，在 Dashboard 點擊 `New +` 按鈕
   - 選擇 `Web Service`

3. **連接 GitHub**
   - 點擊 `Connect a repository`
   - 授權 Render 訪問你的 GitHub（點擊 `Connect` 或 `Authorize`）
   - 在列表中找到 `toeic-vocab-test` 儲存庫
   - 點擊旁邊的 `Connect` 按鈕

4. **配置服務**（Render 應該會自動偵測設定）
   
   確認以下設定正確：
   - **Name**: `toeic-vocab-test`（或你喜歡的名字）
   - **Region**: 選擇 `Singapore` 或離你最近的區域
   - **Branch**: `main`
   - **Runtime**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn main:app`

5. **選擇免費方案**
   - 在頁面下方，**Instance Type** 選擇 `Free`
   - 點擊 `Create Web Service`

6. **等待部署**
   - 部署過程約需 3-5 分鐘
   - 你會看到部署進度和日誌
   - 完成後，頁面頂端會顯示你的網址

## 🛠️ 故障排除

### 問題 1：推送到 GitHub 時要求密碼
**解決方法**：
- GitHub 現在需要使用 Personal Access Token
- 前往：https://github.com/settings/tokens
- 點擊 `Generate new token (classic)`
- 勾選 `repo` 權限
- 複製 token 並在推送時使用（替代密碼）

### 問題 2：網站無法開啟
**檢查步驟**：
1. 在 Render Dashboard 查看部署狀態
2. 點擊 `Logs` 查看錯誤訊息
3. 確認部署完成（顯示綠色 "Live"）
4. 等待 30-60 秒（可能在喚醒中）

### 問題 3：單字沒有載入
**可能原因**：
- PDF 檔案沒有正確上傳到 GitHub
- 檢查 GitHub 儲存庫中是否有 `docs` 資料夾
